# Clínica de Fisioterapia

Sistema completo de gestão de clínica com Node.js, Express e SQLite.

## Estrutura
- `server.js` - Backend (Node/Express)
- `public/` - Frontend (HTML, CSS, JS)
- `clinica.db` - Banco de dados SQLite

## Como rodar
```bash
npm install express sqlite3 body-parser express-session
node server.js
```
Abra em: http://localhost:3000
